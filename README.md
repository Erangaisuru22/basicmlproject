# Flower Classification Using Machine Learning

## Overview

This project demonstrates a machine learning approach to classify iris flowers into three species: Setosa, Versicolor, and Virginica. The classification is performed using four different machine learning algorithms and their performances are compared.

## Objectives

* Learn the machine learning workflow.
* Train and test multiple classification algorithms.
* Compare model accuracy.
* Predict flower species based on flower measurements.

## Algorithms Implemented

1. K-Nearest Neighbors (KNN)
2. Decision Tree Classifier
3. Random Forest Classifier
4. Support Vector Machine (SVM)

## Dataset

The Iris dataset is a popular machine learning dataset containing 150 flower samples with four numerical features:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

Target Classes:

* Setosa
* Versicolor
* Virginica

## Results

The models were trained and evaluated using a train-test split. Accuracy scores were compared to determine the best-performing algorithm for flower classification.

## Conclusion

This project provides a simple introduction to supervised machine learning and classification using Python and Scikit-learn.
